# Телеграм бот погоды
показывает текущую погоду по геолокации пользователя. Используется OpenWeatherMap API.

# Возможности

Получить погоду по текущему местоположению
Узнать температуру, влажность, описание погоды

# Как запустить

1. Установите pyTelegramBotAPI:
   pip install pyTelegramBotAPI requests

2. Откройте файл WeatherTelegramBot.py и укажите свои данные:
   API_KEY = 'ваш_ключ_openweathermap'
   TOKEN = 'ваш_токен_телеграм_бота'

3. Запустите бота

# Как пользоваться

- Отправьте боту команду /start
- Нажмите "Получить погоду" и разрешите доступ к геолокации
- Бот пришлет погоду
