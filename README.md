# WeatherTelegramBot

## Описание проекта

Телеграм-бот для получения текущей погоды по местоположению пользователя. Бот запрашивает геолокацию и отправляет данные о температуре, влажности и описании погоды с использованием OpenWeatherMap API.

## Технологии

- pyTelegramBotAPI
- requests
- OpenWeatherMap API

## Инструкция по запуску

1. Установите pyTelegramBotAPI:
   pip install pyTelegramBotAPI requests

2. Откройте файл WeatherTelegramBot.py и заполните переменные:
   API_KEY = 'ваш ключ openweathermap'
   TOKEN = 'ваш токен телеграм бота'

3. Запустите бота:
   python WeatherTelegramBot.py
